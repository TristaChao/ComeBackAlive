# 遊戲食譜詳細步驟設計文檔 (Recipes Design - Detailed Steps)

這份文件為 `recipes_design.md` 中的每一道食譜提供了詳細的、按部就班的製作流程，作為開發時的具體參考。

---
## 義式料理 (Italian Cuisine)

### **食譜 1：烤牛排**
*   **類型:** 主餐
*   **說明:** 一切的基礎，簡單的烤肉料理。
1.  **烹飪:**
    *   **地點:** 烤台 (Grill)
    *   **步驟 1:** 手持 `生牛肉` 與烤台互動。
    *   **步驟 2:** 等待進度條完成。
    *   **產出:** `烤牛排`。
2.  **上菜:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 3:** 將 `烤牛排` 放在櫃台上。
    *   **步驟 4:** 手持 `盤子` 與櫃台互動，裝盤成 `盤裝烤牛排`。

### **食譜 2：洋蔥湯**
*   **類型:** 湯品
*   **說明:** 醬汁的基礎，也是一道美味的湯品。
1.  **烹煮:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** 手持 `空鍋子` 從「水槽」取 `水`，將 `水鍋` 放在煮鍋上。
    *   **步驟 2:** 手持 `洋蔥` 與煮鍋互動，烹煮後成為 `洋蔥湯`。
2.  **上菜:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 3:** 手持 `碗` 與煮鍋互動，取出 `碗裝洋蔥湯`。

### **食譜 3：蘑菇醬汁牛排**
*   **類型:** 主餐
*   **說明:** 在烤牛排的基礎上增加更豐富的風味。
1.  **準備牛排:**
    *   **步驟 1:** 完成一份 `烤牛排` 並將其放置在「櫃台」上 (無需裝盤)。
2.  **準備蘑菇醬汁:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 2:** 製作一份 `洋蔥湯`。
    *   **步驟 3:** 手持 `蘑菇` 與鍋中的 `洋蔥湯` 互動，烹煮後成為 `蘑菇醬汁`。
3.  **組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 4:** 手持 `蘑菇醬汁` (可用碗盛出) 與櫃台上的 `烤牛排` 互動。
    *   **產出:** `蘑菇醬汁牛排`。
    *   **裝盤:** 最後將 `蘑菇醬汁牛排` 裝盤後即可食用。

### **食譜 4：番茄醬汁牛排**
*   **類型:** 主餐
*   **說明:** 經典的口味組合。
1.  **準備牛排:**
    *   **步驟 1:** 完成一份 `烤牛排` 並將其放置在「櫃台」上。
2.  **準備番茄醬汁:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 2:** 製作一份 `洋蔥湯`。
    *   **步驟 3:** 手持 `番茄` 與鍋中的 `洋蔥湯` 互動，烹煮後成為 `番茄醬汁`。
3.  **組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 4:** 手持 `番茄醬汁` 與櫃台上的 `烤牛排` 互動。
    *   **產出:** `番茄醬汁牛排`。
    *   **裝盤:** 最後將 `番茄醬汁牛排` 裝盤後即可食用。

### **食譜 5：簡易沙拉**
*   **類型:** 開胃菜
*   **說明:** 清爽的開胃菜，是切菜技巧的入門。
1.  **備料 (Chopping):**
    *   **地點:** 砧板 (Chopping Board)
    *   **步驟 1:** `番茄` → 切 → `番茄丁`。
    *   **步驟 2:** `洋蔥` → 切 → `洋蔥丁`。
2.  **裝盤 (Plating):**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 3:** 從「碗盤櫃」拿出 `沙拉碗`，放在櫃台上。
    *   **步驟 4:** 手持 `番茄丁` 與櫃台上的 `沙拉碗` 互動，將其放入。
    *   **步驟 5:** 手持 `洋蔥丁` 與櫃台上的 `裝有番茄丁的沙拉碗` 互動，將其放入。
    *   **產出:** `簡易沙拉` (已在碗中)。

### **食譜 6：羅宋湯**
*   **類型:** 湯品
*   **說明:** 溫暖身心的濃湯，需要多次備料與烹煮。
1.  **備料 (Chopping):**
    *   **地點:** 砧板 (Chopping Board)
    *   **步驟 1:** `牛肉` → 切 → `牛肉塊`。
    *   **步驟 2:** `高麗菜` → 切 → `高麗菜絲`。
    *   **步驟 3:** `番茄` → 切 → `番茄丁`。
2.  **烹煮 (Cooking):**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 4:** 手持 `空鍋子` 從「水槽」取 `水`，將 `水鍋` 放在煮鍋上。
    *   **步驟 5:** 手持 `牛肉塊` 與煮鍋互動。
    *   **步驟 6:** 手持 `高麗菜絲` 與煮鍋互動。
    *   **步驟 7:** 手持 `番茄丁` 與煮鍋互動，烹煮後成為 `羅宋湯`。
3.  **上菜 (Plating):**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 8:** 手持 `碗` 與煮鍋互動，取出 `碗裝羅宋湯`。

### **食譜 7：番茄醬汁義大利麵**
*   **類型:** 主餐
*   **說明:** 經典的義式主食。
1.  **製作番茄醬汁 (Sequential Cooking):**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** 製作 `洋蔥湯`。
    *   **步驟 2:** 在 `洋蔥湯` 中加入 `番茄`，烹煮後成為 `番茄醬汁`。
2.  **烹煮麵條:**
    *   **地點:** 煮鍋 (Stove) - *需要第二個煮鍋*
    *   **步驟 3:** `麵條` + `水` → 煮 → `熟麵條`。
3.  **組合與上菜 (Combination):**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 4:** 將 `熟麵條` 放在櫃台上。
    *   **步驟 5:** 手持 `盤子` 與櫃台上的 `熟麵條` 互動，裝盤成 `盤裝麵條`。
    *   **步驟 6:** 手持 `番茄醬汁` 與 `盤裝麵條` 互動，淋上醬汁。
    *   **產出:** `番茄醬汁義大利麵`。

### **食譜 8：布丁**
*   **類型:** 甜點
*   **說明:** 香甜滑順的飯後甜點。
1.  **混合與烹煮:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** `牛奶` + `雞蛋` → 放入鍋中 → `蛋奶液`。
    *   **步驟 2:** `蛋奶液` + `糖` → 烹煮 → `布丁`。
    *   **取出:** 空手互動即可取出。

## 日式料理 (Japanese Cuisine)

### **食譜 9：烤蔬菜串**
*   **類型:** 開胃菜
*   **說明:** 簡單又美味的日式串燒。
1.  **備料與組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 1:** 從「竹籤盒」拿出 `竹籤` 放在櫃台上。
    *   **步驟 2:** 手持 `蘑菇` 與 `竹籤` 互動 → 產出 `蘑菇串`。
    *   **步驟 3:** 手持 `紅蘿蔔` 與 `蘑菇串` 互動 → 產出 `蔬菜串`。
2.  **烹飪:**
    *   **地點:** 烤台 (Grill)
    *   **步驟 4:** `蔬菜串` → 烤 → `烤蔬菜串`。
3.  **上菜:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 5:** 將 `烤蔬菜串` 與 `盤子` 組合，變成 `盤裝烤蔬菜串`。

### **食譜 10：味噌湯**
*   **類型:** 湯品
*   **說明:** 傳統的日式湯品。
1.  **備料:**
    *   **地點:** 砧板 (Chopping Board)
    *   **步驟 1:** `豆腐` → 切 → `豆腐塊`。
2.  **烹煮:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 2:** `水` + `味噌` → 煮 → `味噌湯底`。
    *   **步驟 3:** `味噌湯底` + `豆腐塊` → 煮 → `味噌湯`。
    *   **取出:** 手持`碗`與煮鍋互動，取出 `碗裝味噌湯`。

### **食譜 11：日式咖哩飯**
*   **類型:** 主餐
*   **說明:** 份量十足、營養均衡的日式國民美食。
1.  **煮飯:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** `米飯` + `水` → 煮 → `熟白飯`。
2.  **備料:**
    *   **地點:** 砧板 (Chopping Board)
    *   **步驟 2:** `馬鈴薯` → 切 → `馬鈴薯塊`。
    *   **步驟 3:** `紅蘿蔔` → 切 → `紅蘿蔔塊`。
    *   **步驟 4:** `肉` → 切 → `肉塊`。
3.  **煮咖哩醬:**
    *   **地點:** 煮鍋 (Stove) - *需要第二個煮鍋*
    *   **步驟 5:** `水` + `馬鈴薯塊` + `紅蘿蔔塊` + `肉塊` → 煮。
    *   **步驟 6:** 加入 `咖哩塊` → 煮 → `咖哩醬`。
4.  **組合上菜:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 7:** 將 `熟白飯` 用 `盤子` 裝盤，放在櫃台上。
    *   **步驟 8:** 手持 `咖哩醬` (用碗或醬汁船從鍋中取出) 與 `盤裝白飯` 互動。
    *   **產出:** `日式咖哩飯`。

### **食譜 12：三色糰子**
*   **類型:** 甜點
*   **說明:** 可愛又美味的日式點心。
1.  **製作糰子:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 1:** `米粉` + `水` → 揉 → `糰子生麵團`。
    *   **步驟 2:** `糰子生麵團` + `糖` → 揉 → `甜味糰子生麵團`。
2.  **烹煮:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 3:** 將 `甜味糰子生麵團` 放入加了水的煮鍋中 → 煮 → `熟糰子`。
    *   *(美術表現上，煮熟後會自動變成三種顏色)*
3.  **組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 4:** 將三個 `熟糰子` 放在櫃台上。
    *   **步驟 5:** 手持 `竹籤` 與櫃台上的 `熟糰子` 互動，將其串起。
    *   **產出:** `三色糰子`。

## 家常菜 (Home-style Cuisine)

### **食譜 13：漢堡排**
*   **類型:** 主餐
*   **說明:** 多汁飽滿的家常主食。
1.  **備料:**
    *   **地點:** 砧板 (Chopping Board)
    *   **步驟 1:** `生牛肉` → 切 → `絞肉`。
    *   **步驟 2:** `洋蔥` → 切 → `洋蔥丁`。
2.  **組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 3:** 將 `絞肉` 放在櫃台上，手持 `洋蔥丁` 與之互動 → `生漢堡排`。
3.  **烹飪:**
    *   **地點:** 烤台 (Grill)
    *   **步驟 4:** `生漢堡排` → 烤 → `漢堡排`。
    *   **裝盤:** 將 `漢堡排` 與 `盤子` 組合後即可食用。

### **食譜 14：豪華蔬菜串**
*   **類型:** 開胃菜
*   **說明:** 自由組合喜歡的蔬菜。
1.  **備料:**
    *   **地點:** 砧板 (Chopping Board)
    *   `蘑菇` → `蘑菇片`；`番茄` → `小番茄塊`；`洋蔥` → `洋蔥塊`。
2.  **串上食材 (多次組合):**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 1:** 將 `竹籤` 放在櫃台上。
    *   **步驟 2:** 依序手持 `蘑菇片`, `小番茄塊`, `洋蔥塊` 與櫃台上的竹籤互動。
    *   **產出:** `豪華蔬菜串`。
3.  **烹飪:**
    *   **地點:** 烤台 (Grill)
    *   `豪華蔬菜串` → 烤 → `烤豪華蔬菜串` (需裝盤)。

### **食譜 15：牧羊人派**
*   **類型:** 主餐
*   **說明:** 需要耐心與多樣技巧的終極料理。
1.  **準備內餡 (序列料理):**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** `絞肉` (牛肉切碎) + `洋蔥丁` + `水` → 煮 → `肉醬底`。
    *   **步驟 2:** `肉醬底` + `番茄丁` → 煮 → `牧羊人派內餡`。
2.  **準備馬鈴薯泥:**
    *   **地點:** 煮鍋 (Stove)
    *   **步驟 1:** `馬鈴薯` + `水` → 煮 → `熟馬鈴薯`。
    *   **地點:** 櫃台 (Counter)
    *   **步驟 2:** 將 `熟馬鈴薯` 放在櫃台，空手互動 → `馬鈴薯泥`。
3.  **組合:**
    *   **地點:** 櫃台 (Counter)
    *   **步驟 1:** 將 `牧羊人派內餡` 用 `碗` 盛裝，放在櫃台上。
    *   **步驟 2:** 手持 `馬鈴薯泥` 與 `碗裝內餡` 互動。
    *   **產出:** `牧羊人派`。
