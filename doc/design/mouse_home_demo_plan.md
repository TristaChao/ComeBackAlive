# 遊戲 Demo 流程設計：鼠窩裡的生活

這份文件詳細描述了一個以鼠窩為背景的遊戲 Demo 流程，涵蓋了對話、道具管理、烹飪與裝潢等核心機制。

---

## 1. NPC 互動：與鼠媽對話

**目標：** 玩家與 NPC（鼠媽）進行多層次的對話互動。

1.  **觸發對話：**
    *   玩家角色靠近「鼠媽」。
    *   玩家按下 `E` 鍵。
    *   畫面上出現對話框。

2.  **第一層對話選項：**
    *   **標題：** "要聊聊什麼嗎？"
    *   **選項 1：** 聊聊料理
    *   **選項 2：** 聊聊家裡
    *   **選項 3：** 聊聊外面

3.  **第二層對話選項（選擇「聊聊料理」後）：**
    *   **選項 1：** 牛排
    *   **選項 2：** 烤紅蘿蔔

4.  **最終對話（選擇「牛排」後）：**
    *   顯示文字："把牛肉直接拿去烤台上烤熟就好了。"
    *   對話結束。

---

## 2. 道具與庫存系統

**目標：** 玩家能管理手持物品、頰囊（背包）與外部倉庫。

### 2.1 頰囊系統 (背包)

*   **開啟/關閉：**
    *   玩家按下 `X` 鍵可開啟「頰囊視窗」（一個由空格子組成的介面）。
    *   按下 `esc` 鍵可關閉視窗。
*   **視覺表現：**
    *   當頰囊中有物品時，角色頰囊會「鼓起」。
    *   頰囊清空後，恢復原狀。

### 2.2 物品互動：拾取、存放與取出

1.  **拾取物品：**
    *   玩家靠近「櫃台」。
    *   按下 `E` 鍵，角色拾取「洋蔥」。
    *   **狀態：** 角色動畫顯示手持洋蔥。

2.  **存入頰囊：**
    *   在手持洋蔥的狀態下，玩家按下 `X` 鍵。
    *   洋蔥從手上消失，存入頰囊。
    *   **狀態：** 角色頰囊鼓起。此時打開頰囊視窗，可以看到洋蔥圖示。

3.  **從頰囊取出：**
    *   打開頰囊視窗。
    *   選取「洋蔥」。
    *   按下 `E` 鍵，角色將洋蔥從頰囊吐出至手上。
    *   **狀態：** 角色恢復手持洋蔥的動畫，頰囊恢復正常。

### 2.3 倉庫系統

1.  **進入倉庫：**
    *   玩家靠近「倉庫」，畫面上出現提示文字 `（E）倉庫`。
    *   按下 `E` 鍵，開啟「倉庫視窗」。

2.  **倉庫介面：**
    *   **左側（玩家物品）：**
        *   **上方：** 顯示「目前手持物品」（例如：洋蔥）。
        *   **下方：** 顯示「頰囊視窗」的內容（目前為空）。
    *   **右側（倉庫物品）：**
        *   顯示倉庫內的物品（例如：一個牛肉、一個蘑菇）。

3.  **物品轉移：**
    *   玩家將手上的「洋蔥」放入右側的倉庫。
    *   玩家從右側倉庫中選取並取出「牛肉」。
    *   退出倉庫視窗。
    *   **狀態：** 角色變為手持「牛肉」。

---

## 3. 烹飪系統：烤牛肉

**目標：** 玩家利用場景物件（烤台）來改變物品狀態。

1.  **放置食材：**
    *   玩家手持「牛肉」靠近「烤台」。
    *   按下 `E` 鍵，將牛肉放在烤台上。
    *   （可選）玩家可先將牛肉放在「櫃台」上，再拿起並放到烤台。

2.  **烹飪過程：**
    *   **第一階段（烤熟）：**
        *   牛肉放到烤台上後，上方出現「進度條」。
        *   進度條跑完後，「牛肉」變為「牛排」。
    *   **第二階段（烤焦）：**
        *   變成牛排後，若不取下，會再次出現一條新的進度條。
        *   第二條進度條跑完後，「牛排」變為「燒焦的東西」。

---

## 4. 裝潢模式

**目標：** 玩家可以重新佈置場景中的家具。

1.  **進入模式：**
    *   玩家按下 `Y` 鍵。
    *   跳出確認視窗："進入裝潢模式？"
        *   **選項：** `確定` / `取消`
    *   **前置條件：** 若玩家手上有物品，則無法進入，並提示 "放下東西才能裝潢"。
    *   玩家選擇 `確定` 進入裝潢模式。

2.  **移動家具：**
    *   在裝潢模式下，玩家靠近「櫃台」，出現提示 `(E)櫃台`。
    *   按下 `E` 鍵「拿起」櫃台。此時櫃台物件會跟隨玩家移動，或顯示為半透明的預覽位置。
    *   玩家移動到新位置後，再次按下 `E` 鍵「放下」櫃台。
    *   此機制同樣適用於「倉庫」及其他台子等家具。
