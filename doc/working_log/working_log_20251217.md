### 已完成的目標 (Completed Goals):

1.  **NPC 互動系統**
    *   已完成與鼠媽的多層次對話功能，包含靠近時的提示文字顯示/隱藏邏輯。

2.  **烹飪系統 (完整)**
    *   **櫃台 (Counter):** 已實作成一個物品容器，可與玩家交換手持物品，並具備視覺回饋。
    *   **烤台 (Grill):** 已完整實現三段式烹飪循環（生 → 熟 → 燒焦），包含進度條、計時器、狀態改變，以及根據不同狀態顯示不同物品圖示的視覺回饋。

3.  **道具與庫存系統 (大部分完成)**
    *   **頰囊系統 (Pouch/Backpack):**
        *   可按 `X` 鍵開啟頰囊 UI，按 `esc` 鍵關閉。
        *   UI 會以格狀顯示頰囊中的物品圖示。
        *   UI 中可用方向鍵移動選擇框，被選中的物品會放大以提供視覺回饋。
        *   可按 `E` 鍵將選中的物品取出到手上。
        *   將手持物品放入頰囊的功能已完成。
    *   **倉庫系統 (Warehouse):**
        *   已建立世界中的倉庫物件，玩家可靠近互動以打開倉庫 UI。
        *   倉庫 UI 具備左右分割畫面（左:頰囊，右:倉庫），並能正確顯示雙方的物品。
        *   已實現使用 `Tab` 鍵在左右兩個面板間切換焦點的功能。
        *   已實現使用方向鍵在當前有焦點的面板內移動選擇框的功能。
        *   已實現按 `E` 鍵在頰囊與倉庫之間轉移物品的功能。
        *   已實現當目標容器已滿時，會顯示「放不進去」或「拿不出來」的提示訊息。
